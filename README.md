## Deploy to Vercel

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Navigate to project folder:
```bash
cd homework-helper-react
```

3. Install dependencies:
```bash
npm install
```

4. Deploy:
```bash
vercel
```

Follow the prompts, and your app will be live!
